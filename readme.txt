--description
this program will implement a simulation of the Disk file system. the program is a simulation of the environment in software for events and actions that occur 
in a real system. the program will show a way of organizing the hard disk and saving information on the files.
we have 3 classes each class has its job in the program :
1. FsFile:- is a data structure that stores basic information about the file using the index allocation method
--functions
1. FsFile():-constructor
2.getfile_size:- gets the file size
3.setfile_size:- sets the file size
4.getBlock_in_use:- gets the numbre of blocks in use 
5.setBlock_in_use:- sets the numbre of blocks in use 
6.getIndex_block:- gets the place of the index block 
7.setIndex_block:- sets the place of the index block 

2. FileDescriptor:-  this class contains information about the name of the file and a pointer to the file of the file and if the file is in use or not. 
-----functions 
1. fileDescriptor():-constructor
2.getFileName:- gets the file name
3.getInUse:- gets if the file is in use 
4.setInUse:- sets if the file is in use 
5.getFsFile:- gets the fsfile of the file 
6.setFsFile:- sets the fsfile of the file

3.fsDisk:- this class is a data structure that will do all the oprations for the disk and saves all information for the disk 
-----functions 
1. fsDisk():-constructor
2.getEmptyPlaceInDisk:- will go and find the next empty place in disk and return its number
3.howManyRemaning:- for a file this function will go get how many remaning places we can wright on 
4. listAll:- will print the disk and all the file discreptors 
5.fsFormat:- will format the disk and will set the number of blocks and the block size 
6.CreateFile:- will create and open new file and like it with its file discrptor 
7. OpenFile :- checks if the file is closed and then opens it 
8.  CloseFile:- will check if the file is open then close it 
9. WriteToFile :- will wright a string on a file in the disk 
10. ReadFromFile:- reads on a file in the disk 
11. DelFile:- will delete a file from the disk
